# Indy CM Towing & Rescue

Professional towing company website built with Next.js, TypeScript, and Tailwind CSS.

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

This project is ready for Vercel. Import the repository in Vercel and use the default Next.js settings.

Set this environment variable in Vercel after you choose the live website URL:

```bash
NEXT_PUBLIC_SITE_URL=https://indycmtowing.com
```

## Notes

- The site uses the real logo and tow truck photos in `public/assets`.
- The contact form prepares a text message with tow details for faster mobile requests.
